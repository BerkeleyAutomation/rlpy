Detailed Description

Developed by N.

Kemal Ure Dec 3rd 2012 at MIT # Edited by Alborz Geramifard on Feb 20 2013 #

State is : Location of Agent_1 x ... x Location of Agent n... Location of Intruder 1 x ...x Location of Intruder_m n is number of agents, m is number of intruders Location is 2D position on a grid Each agent can move in 4 directions + stay still, there is no noise Each intruder moves with a fixed policy (specified by the user) By Default, intruder policy is uniform random Map of the world contains fixed number of danger zones, Team receives a penalty whenever there is an intruder on a danger zone in the absence of an agent Task is to allocate agents on the map so that intruders do not enter the danger zones without attendance of an agent
