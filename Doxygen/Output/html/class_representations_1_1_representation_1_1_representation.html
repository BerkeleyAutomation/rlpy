Detailed Description

The Representation class describes the information the Agent gathered about the Domain.

Representation provides the basic framework for Representations to interact with the Agent and Policy Classes. All new Representation implimentations should inherit from Representation.

Note
It is assumed that the Linear Function approximator Family is being used.
Author
Developed by Alborz Geramiard Oct 25th 2012 at MIT

Given a state it returns a vector with the same dimensionality of s each element of the returned valued is the zero-indexed bin number corresponding to s This function accepts scalar inputs when the domain has 1 dimension CompactBinary version exclude feature activation for the negative case of binary features.

For example if the light is off, no feature corresponds to this case and hence nothing is activated.
